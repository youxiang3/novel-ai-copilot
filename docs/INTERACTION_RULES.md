# 交互规则

## 作品管理弹窗

- 作品卡片和列表视图中的“作品管理”应打开 WorkManagementModal。
- 更多菜单中的“重命名”“设置目标字数”“设置发布平台”可以打开同一弹窗并聚焦相关字段。
- 弹窗可编辑：作品名、笔名、项目状态、当前发布状态、目标字数、计划章节数、本周更新目标、发布平台、简介和标签。
- 保存后应同步刷新作品库卡片、列表视图、右侧当前作品详情、顶部统计、搜索 / 筛选 / 排序结果和作品总览。
- 数字字段必须做非负整数兜底，不能展示 NaN。
- 保存仍是前端本地持久化，写入 `localStorage:yixie-works-library-v1`。

## 作品总览展示

- 作品总览必须读取作品库同一份当前作品数据。
- 字段缺失时显示友好默认值，不展示 `undefined`、`null` 或 `NaN`。
- 当前字数和章节数优先使用现有统计字段，不在总览页重写复杂统计逻辑。
- 刷新浏览器后，总览应从 `localStorage:yixie-works-library-v1` 恢复最新作品数据。
- 总览展示的是本地项目驾驶舱，不代表后端已保存。

## 全局大纲展示

- 全局大纲字段为 `globalOutline`。
- 默认展示前 4 条。
- 超过 4 条时显示“展开全部 / 收起”。
- 没有全局大纲时显示：“还没有全局大纲，可在作品资料或新建作品确认流程中补充。”
- 可以提供“进入故事线图”入口，但不要在总览中重写故事线图逻辑。
- 本阶段不做复杂大纲编辑器。

## 游客草稿 / 正式作品提示

- 游客草稿必须提示：仅保存在当前浏览器，本地缓存清理或更换设备后可能丢失。
- 游客草稿不得显示为云端正式作品。
- 正式作品可以提示：已加入作品库，当前以本地保存为准。
- 云端同步未完成时，不得显示“云端已同步”这类确定性文案。

## 官方工具状态

官方工具应展示两类状态：

- 功能状态：未启用、已启用、进行中、上次通过、存在风险。
- 接入状态：原型展示、模型待接入、本地可用、已接入模型、云端待同步。

规则：

- 未选择作品时，工具按钮显示“选择作品后启用”。
- 选择作品后，未启用工具显示“启用”。
- 已启用工具显示“管理”或“查看”。
- 进行中工具显示“进行中”。
- 检查类工具显示上次检查状态。
- 长篇记忆、OOC 检查和真实模型相关工具如未接入真实服务，必须显示“原型展示”或“模型待接入”。

## 同步状态展示

同步状态包括：

- 仅本地
- 待同步
- 同步中
- 已同步
- 同步失败

当前同步状态主要是前端字段和 UI 占位。除非后端同步服务完成，否则不要在文案中承诺云端已保存。

## 危险动作后续规则

导出作品当前已开放 Markdown / JSON 本地导出，仅用于阅读和备份，不会删除或修改当前作品。JSON 备份可从作品库“导入备份”入口恢复为新的本地作品，但不是云端备份。

导出中心当前支持 TXT、Markdown、DOCX、EPUB 本地文件导出；PDF 使用浏览器打印版，由用户在打印面板中另存为 PDF。DOCX / EPUB 当前为基础格式，不承诺复杂排版、封面嵌入或高级目录。

复制、归档、删除当前已在本地闭环内开放：

- 复制作品：二次确认后创建新的本地副本，副本标记为“仅本地保存”，不继承云端同步状态。
- 导入备份：读取本产品导出的 JSON 文件，创建新的本地作品，不覆盖原作品。
- 归档作品：确认后设置为已归档，不删除作品数据。
- 删除作品：两次确认后删除当前浏览器中的本地作品数据。
- 所有本地数据动作不得重命名、清空或迁移 localStorage key。

仍未开放：

- 云端备份。
- 后端恢复。
- 跨设备恢复。

## 空状态和异常兜底

- 空字符串、空数组、缺失字段使用明确空状态文案。
- 数字字段缺失时使用 0 或“未设置”，不展示 NaN。
- 发布平台缺失时显示“未设置平台”。
- 同步状态缺失时，游客草稿显示“仅本地”，正式作品显示本地保存或待同步占位。
- 资料摘要缺失时显示“暂无资料”，不阻断页面渲染。
- 工具状态缺失时使用默认原型状态。
