## 0.1 历史或参考作品引入

为了让作者在创作初期就能有明确参考，同时保持创作自主权：

### 引导提问方式

在情绪、题材确定之后，AI会问作者：

> “你希望故事参考哪些背景、历史事件、电影或小说风格？ 你可以选择单一或多种来源，也可以自由描述。”

示例选项（可多选）:

A. 历史事件（如秦朝、冷战、文艺复兴等）

B. 电影（如《黑客帝国》、《盗梦空间》）

C. 小说（如《三体》、《鬼吹灯》、《盗墓笔记》）

D. 同类型网文作品（如系统流、无限流、高武、玄幻等）

E. 自定义画面/情绪参考（自由描述）

### 工作流效果

- AI 接住作者提供的参考素材。
- 提取关键元素：场景、逻辑、节奏、情绪、冲突。
- 将参考元素自动映射到 Workflow 节点，辅助生成剧情、世界观、角色设定。
- 保留作者自主选择权，AI 不直接替作者决定，而是提供选项和追问。

### 产品文档更新

文档中“创作启动模式”和“对话式引导系统”部分新增：

- 历史/作品/题材参考提问节点。
- 引导节点在 Workflow 中形成一个独立分支，输入作者的参考信息后，会被映射到剧情生成与世界状态更新中。
- Prompt Export Layer 自动记录参考来源，供多模型生成使用，保持风格一致性或多版本对比。

这保证了：

1. 作者始终主导题材和参考背景。
2. AI 根据作者参考进行结构化引导和生成。
3. 可直接投喂多模型，输出不同风格版本。
4. 历史/作品参考成为 Workflow 节点，形成长期记忆与伏笔池的一部分。



## 0.2 引导提问作为工作流节点

在 NovelCraft 中，所有 AI 提问本身被设计为 Workflow 节点，形成：

### Narrative Conversation Workflow Engine

- 每一次 AI 提问都是一个 Node（节点）
- Node 输入：作者上一次回答 + 参考素材 + 当前剧情状态 + 风格参数
- Node 输出：提示下一步创作方向或下一轮追问
- Node 状态流转：成功接住作者回答，更新剧情/世界状态，推进到下一个节点
- Node 可中断恢复：允许作者随时修改选择，重新计算后续节点输出
- Node 自动决策：对选择模糊的回答提供默认选项或多选建议

### 引导提问工作流示例

1. **情绪钩子选择 Node**：让作者确定核心情绪（揭秘感、压迫感等）
2. **题材/背景 Node**：作者决定小说题材和初始世界画面
3. **参考素材 Node**：作者提供历史事件、电影、小说或同类型作品参考
4. **主角认知滤镜 Node**：作者选择主角独有能力或视角
5. **世界异常 Node**：基于题材和参考素材，AI引导作者定义第一个异常
6. **章节钩子 Node**：AI引导作者确认第一章关键冲突和情绪峰值
7. **Prompt 导出 Node**：生成可喂给不同模型的结构化 Prompt

### 特点

- 每一个问题都是工作流节点
- 输出不仅是回答，还会更新 Narrative Ledger（黄金账本）和 Foreshadowing Pool（伏笔池）
- 支持多模型对比生成
- 保证作者主导创作，AI提供结构化引导

这样设计可以让 NovelCraft 不仅是 AI 写作工具，而是一个真正的：

> AI Narrative OS + 引导式创作 Workflow Engine

## 新增补充点：网文 AI 智能体功能覆盖

### 1. 作者主导 + AI 引导

- 明确作者主导故事主题、题材、历史或参考素材
- AI 提问和引导形成完整 Workflow 节点
- AI 不代写作者决定，保持作者创作掌控权
- Workflow节点包括：情绪节点、题材节点、参考素材节点、主角认知滤镜节点、世界异常节点、章节冲突节点、Prompt导出节点、伏笔池更新节点

### 2. 历史 / 参考作品 / 同类型小说支持

- 引导作者选择参考历史事件、电影、小说或同类型网文
- AI 自动提取关键元素映射到剧情和世界观设计
- Prompt Export Layer 支持生成可投喂多模型的结构化Prompt

### 3. 多模型支持与对比生成

- Workflow推进后，可同时生成 GPT / Claude / DeepSeek / Gemini 多模型版本
- 支持文风对比、情绪对比、节奏对比
- 作者可挑选最终风格，增加创作掌控感

### 4. 长文本记忆与世界状态维护

- 黄金账本（Narrative Ledger）跟踪物理与心理状态
- 伏笔池（Foreshadowing Pool）跟踪未填坑的情节线索
- 自动回收旧伏笔和生成新伏笔节点

### 5. 节奏与情绪工业化管理

- 章节冲突和爽点密度分析
- 爽点与节奏数据库，预测追读率
- 名场面识别和Epic Scene Engine
- 平台适配策略 Engine（不同平台优化节奏和情绪）

### 6. AI 共创与协作模式

- 灵感模式 / 编辑模式 / 共创模式 / 代写模式
- 支持长期连载，作者可选择AI协作强度
- 自动学习作者偏好，形成 Author Narrative DNA

### 7. 对话引导本身作为 Workflow

- 每一次提问都是节点，形成完整工作流
- 节点输入：作者回答 + 历史/参考素材 + 当前状态 + 风格参数
- 节点输出：下一步创作方向或追问
- 节点状态流转与中断恢复，保持长期连载一致性

### 8. 网文类型与平台优化覆盖

- 支持快节奏网文（番茄）、长篇网文（起点/晋江）、轻小说/短剧/漫画改编
- 每个平台有特定的节奏、钩子、章节长度优化策略
- AI根据作者选择调整Prompt和节奏滑条，自动生成章节初稿或结构化提示
