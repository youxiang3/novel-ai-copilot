# 📖 NovelAI Copilot - 核心产品文档 (PRD)

## 1. 产品概述与定位

- **产品名称**：NovelAI Copilot (网文全链路 AI 副驾)
- **核心目标**：面向网文作者及工作室的“长篇小说创作与IP孵化工作站”，解决长文本创作卡文、上下文大模型记忆丢失、以及小说短剧/漫剧衍生痛点。
- **项目形态**：前后端分离的现代化 Web SPA 应用。

## 2. 核心架构（三大引擎）

1. **文笔克隆引擎**：提取作者过往文字特征，生成专属 System Prompt，保证 AI 续写风格一致。
2. **动态分层记忆引擎**：拆分长期记忆（大纲/人物）、中期记忆（章节摘要）、短期记忆（近期正文），按需组装 Prompt，用极低 Token 成本突破几百万字长文本限制。
3. **短画面转正文引擎**：用户输入几句短画面描述，系统结合上下文记忆流，自动扩写成 1000-2000 字包含细节的正文章节。

## 3. 核心名词解释 (Glossary)

- **Lore (设定/词条)**：包含人物卡、地点、物品、宗门等结构化世界观设定库。
- **Summary (摘要)**：每写完一章，后台静默 Agent 自动生成的 200 字客观剧情总结，用于拼接上下文。
- **Scene-to-Chapter (短画面扩写)**：用户的核心输入方式，通过自然语言驱动小说正文生成。。
